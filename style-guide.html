Business of Connections — Style Guide

The shared visual language for the Business of Connections website. Use it when building new subpages so everything looks like one site.

It matches the original Framer design system — based primarily on the home page and secondarily on Business of Competition (/bocomp26).

Unlisted page · not in site navigation

Typography

Erode for headings, DM Sans for everything else. Headings are Black; body uses 70% Black. Each specimen renders the same line so styles compare like-for-like.

Pairing rule: Erode (serif) for headings and the logo wordmark. DM Sans (sans-serif) for everything else — body, labels, buttons, UI, card content.
